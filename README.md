# 기술 교육 미니 교수학습 단원 (Technology Education Mini Instructional Unit)

본 프로젝트는 대학원 과정 "기술 교육" 수업을 위해 제작된 교육 웹사이트입니다. 단순한 웹 기능을 넘어 교수 설계 원리를 충실히 반영하여 설계되었습니다.

## 🔗 라이브 데모 (GitHub Pages)
GitHub Pages를 통해 호스팅된 사이트를 확인하실 수 있습니다:
> **[기술 교육 사이트 바로가기](https://qkrwlgns9813.github.io/sample/)**
> *(참고: GitHub Settings > Pages에서 확인된 실제 URL로 수정하여 사용하세요.)*

## 📚 프로젝트 개요
- **대상:** 중학교 기술·가정 교과 학습자 및 예비 교사
- **목적:** 기술의 기본 개념 이해 및 형성 평가를 통한 자기주도적 보충 학습 지원
- **교수 설계 흐름:** 도입 → 개념 학습 → 형성 평가 (OX 퀴즈) → 피드백 및 보충 학습 → 차시 예고

## ✨ 주요 특징
- **학술적 디자인:** 교직 실무 및 대학원 과제에 적합한 깔끔하고 신뢰감 있는 디자인 (style.css)
- **형성 평가 시스템:** 5문항의 OX 퀴즈를 통해 학습 성취도를 즉각적으로 점검 (script.js)
- **개별 맞춤형 피드백:** 득점에 따른 진단 메시지와 틀린 문항에 대한 보충 설명(Remediation) 제공
- **반응형 설계:** 모바일 및 데스크톱 환경 모두에서 최적화된 가독성 제공

## 📁 파일 구조
- `index.html`: 단원 개요, 학습 목표 및 수업 구성 안내
- `lesson.html`: 본차시 학습 내용, OX 퀴즈 및 피드백 섹션
- `style.css`: 교육용 테마 디자인 및 레이아웃 가이드
- `script.js`: 퀴즈 로직 및 동적 피드백 생성 엔진

## 🚀 로컬 실행 방법
1. 본 리포지토리를 클론하거나 파일을 다운로드합니다.
2. 브라우저에서 `index.html` 파일을 엽니다.
